# Submission Checklist

- Public GitHub repository
- Apache-2.0 license detected by GitHub
- CI passing on `main`
- README quickstart works on a clean clone
- `examples/` contains generated report and write-back dry-run artifacts
- Demo video under 3 minutes
- DataHub live demo path verified with Docker
- Devpost submission links to the repository and demo video

Before submitting:

```bash
make test
make examples
make demo
```
